# Scripts for easy foaming

A collection of personal utility scripts for working with OpenFOAM more efficiently.

## About

These are helper scripts I've developed to streamline my OpenFOAM workflow. They handle common tasks like residual plotting, log file analysis, and case management. Feel free to use and adapt them for your own work!


## Requirements

- OpenFOAM (any version)
- Python 3.x
- gnuplot (for plotting scripts)


## Usage

Each script has its own README in its directory with detailed usage instructions.

These are personal scripts, but suggestions and improvements are welcome! Feel free to open an issue or submit a pull request.
